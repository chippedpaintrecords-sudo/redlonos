Instructions
How to set up RedlonMatrix™ for Windows.

1. What it is
A PowerShell-based launcher / UI environment for Windows that runs scripts and mimics an OS.

2. Requirements
Windows 10/11
PowerShell (version X.X)
[Any dependencies]

3. Download & Extract
Download the ZIP
Extract to C:\redlonOS (or anywhere)

4. Running redlonOS
Right-click → “Run with PowerShell”
Or double-click start.bat
Or run .\redlonOS.ps1

5. Permissions
If PowerShell execution policy blocks it:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

6. Optional: Create a Shortcut
Right-click → New → Shortcut
Target: powershell.exe -ExecutionPolicy Bypass -File "C:\redlonOS\redlonOS.ps1"

7. Updating
Replace folder contents with new version
Keep settings.json if you want persistence
