What is RedlonMatrix™?
RedlonMatrix™ is a text-based operating system that runs through powershell scripts.
    


Why?

Because I felt like it. This can also be installed on a dedicated user account.

RedlonMatrix™ is:

• Low-resource
• Easy to use
• Customizable
